常用服务

围绕真实业务场景补齐调用、日志、安全和充值能力。

API Key

创建密钥、设置 scopes、设置过期时间，并支持启用、停用和删除。

调用日志

直接读取任务结果表，查看任务状态、耗时和时间分布。

Webhook

配置异步回调地址、事件类型和签名密钥。

账单与配额

支持充值订单、积分变化、套餐展示和白名单安全控制。

接入流程

从文档到接入，只保留真正对调用方有价值的信息。

步骤一：阅读 `/docs` 中的公开接口说明与代码示例。
步骤二：调用 `/turnstile` 提交任务并获取 `taskId`。
步骤三：轮询 `/result` 获取令牌结果与耗时信息。
步骤四：登录 `/login` 进入统一账户中心，管理密钥、Webhook、白名单和账单。
